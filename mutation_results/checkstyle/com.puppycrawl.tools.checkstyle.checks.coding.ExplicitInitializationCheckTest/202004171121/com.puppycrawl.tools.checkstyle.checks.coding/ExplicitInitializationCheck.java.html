/**

 * <p>

 * </p>
36

37 
 * Rationale: Each instance variable gets
38 
 * initialized twice, to the same value. Java

 * initializes each instance variable to its default
40

41 
 * initialization specified in the code.
42

 * </p>

 * <li>

 * </li>

 * </ul>

 * <p>

59 
 * </p>

64 
 * </pre>

67
